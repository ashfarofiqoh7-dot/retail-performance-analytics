# retail-performance-analytics
Sales performance analysis dashboard for Zara retail operations using Looker Studio.
##Project Overview
This Project analyzes retail sales performance on seasonality, product placement, and promotion effectiveness using Looker Studio.

##Key Strategic Recommendations
* **Relocate Top Products:** Move Top 5 items to the **Front Area** to boost its low performance (6.3 M sales).
* **Scale Promotions:** Expand promos as they drive **2x higher volume** (1.4K vs 870).
* **Boost Seasonal Stock:** Increase stock for "Seasonal Heroes" to capture **48.2% of total revenue**

##Dataset
* **Raw Data:** Original dataset from kaggle.
* **Processed Data:** Cleaned data with calculated Revenue and seasonal categories.
